This is a project to collect multiple open datasets about Austin, TX and put them in an easy-to-use format.

More information is in the doc/ directory.  Start with "project_plan.md" or "project_plan.html".

Directory layout:

    * doc/ Documentation
    * downloads/ This is where large downloaded files are put
    * inputs/    This holds pre-downloaded data files 
    * outputs/   This is where the resulting data is stored
    * src/       This holds the python code
    * tmp/       This holds temporary files being worked on
